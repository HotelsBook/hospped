## 🚀 Como atualizar o site

Este site é estático (HTML/CSS/JS puro) e publicado via GitHub Pages.

### Para editar conteúdo:
1. Edite os arquivos `.html` diretamente
2. Commit e push para a branch `main`
3. O deploy automático ocorre em ~2-5 minutos via GitHub Actions
